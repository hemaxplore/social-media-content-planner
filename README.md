## 👩‍💻 About This Project

A **high-fidelity, minimalist Content Planning Dashboard** designed for social media managers and creators. This professional tool allows you to organize post ideas, schedule dates, and track production status with a sleek "Dark Corporate" aesthetic.

> 🎯 Built with pure HTML, CSS, and Vanilla JavaScript — focusing on performance, clean UI, and robust state management.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Glassmorphism UI** | Premium dark theme with translucent cards and vibrant gradients |
| 🚀 **Splash Screen** | Professional animated entry experience (session-aware) |
| 👯 **Duplicate Plan** | One-click duplication to quickly scale your content ideas |
| ⚠️ **Inteligent Badges** | Auto-calculated "Overdue", "Today", and "Upcoming" status markers |
| 📝 **Content Notes** | Integrated caption idea and notes field for every plan |
| 🔍 **Advanced Filtering** | Live search by keyword, platform, and production status |
| 📱 **Fully Responsive** | Optimized layouts for Mobile, Tablet, and Desktop |
| 💾 **Persistent Storage** | Reliable localStorage integration — your plans never disappear |

---

## 🤖 Integrated Components

### 💫 Splash Screen
A session-aware animated entry screen that welcomes the user with a pulsing brand mark. It uses `sessionStorage` to ensure it only plays once per browser session for the best user experience.

### 📅 Smart Date Tracking
The app automatically compares your planned dates with the current time to provide instant visual feedback on your schedule:
- **Overdue**: Missing the deadline (Red)
- **Today**: Active tasks (Amber)
- **Upcoming**: Future plans (Indigo)
